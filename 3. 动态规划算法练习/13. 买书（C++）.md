## 买书（C++）

编程题

### 题目描述：

小明手里有n元钱全部用来买书，书的价格为10元，20元，50元，100元。
问小明有多少种买书方案？（每种书可购买多本）

### 输入格式：

一个整数 n，代表总共钱数。

### 输出格式：

一个整数，代表选择方案种数。

### 数据范围：

0≤N≤1000。

### 输入样例：

```
20
```

### 输出样例：

```
2
```

任务与要求

运行程序，使其通过所有检查

# 题解
```c++
#include <stdio.h>
#include <algorithm>

using namespace std;

int v[5]={0,10,20,50,100};
int f[5][1010];

int main(){
    int n;
    scanf("%d",&n);
    f[0][0]=1; //初始化 什么都不选也是方案
    for(int i=1;i<=4;i++){
        for(int j=0;j<=n;j++){
            f[i][j]=f[i-1][j]; //不选第i个物品的方案

            for(int k=1;k*v[i]<=j;k++){

                if(k*v[i]<=j){
                    f[i][j]+=f[i-1][j-k*v[i]]; //选k次第i个物品的方案
                }
            }
        }
    }
    printf("%d",f[4][n]);
    return 0;
}
```
