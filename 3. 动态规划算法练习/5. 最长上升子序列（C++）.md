## 最长上升子序列（C++）

编程题

### 题目描述

给定一个长度为 N 的数列，求数值严格单调递增的子序列的长度最长是多少。

### 输入格式

第一行包含整数 N。

第二行包含 N 个整数，表示完整序列。

### 输出格式

输出一个整数，表示最大长度。

数据范围：1≤N≤1000，−109≤数列中的数≤109

### 输入样例：

```
7
3 1 2 1 8 5 6
```

### 输出样例：

```
4 
```

任务与要求

运行程序，使其通过所有检查

# 题解
```c++
#include <iostream>

using namespace std;

const int N = 1010;

int n;
int w[N], f[N];

int main() {
    cin >> n;
    for (int i = 0; i < n; i++) cin >> w[i];

    int mx = 1;    // 找出所计算的f[i]之中的最大值，边算边找
    for (int i = 0; i < n; i++) {
        f[i] = 1;    // 设f[i]默认为1，找不到前面数字小于自己的时候就为1
        for (int j = 0; j < i; j++) {
            if (w[i] > w[j]) f[i] = max(f[i], f[j] + 1);    // 前一个小于自己的数结尾的最大上升子序列加上自己，即+1
        }
        mx = max(mx, f[i]);
    }

    cout << mx << endl;
    return 0;
}
```
