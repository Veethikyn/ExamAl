## 求“水仙花数”

编程题

### 题目描述

“水仙花数”是指一个三位数，其每位数字的立方和等于该数本身。
例子：
153是一个“水仙花数”，因为`153=1*1*1＋5*5*5＋3*3*3`。
编程找出 100-1000 以内的所有水仙花数。

### 输出样例：

```
100-1000的“水仙花数”有：
153
370
371
407
```

任务与要求

运行程序，使其通过所有检查

# 答案
```c
#include <stdio.h>
int main()
{
    int i,j,k,n;
    printf("100-1000的“水仙花数”有：\n"); 
    for(i = 100;i < 1000;i++)
    {
        j = i%10;
        k = i/10%10;
        n = i/100;
        if(j*j*j + k*k*k + n*n*n == i)
            printf("%5d\n",i);
    }
    return 0;
}
```
