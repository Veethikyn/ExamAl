## 求解三位数

编程题

### 题目描述

有一个三位数，个位数字比百位数字大，百位数字又比十位数字大，并且个位数字之和等于各位数字相乘之积，设计一个算法用穷举法求此三位数。

### 输出样例：

```
结果为
213
```

任务与要求

运行程序，使其通过所有检查

# 答案
```c
#include<stdio.h>
void solve()
{	int a,b,c;
	for(a=1;a<=9;a++)
	for(b=0;b<=9;b++)
	for(c=0;c<=9;c++)
	{	if(c>a&&a>b&&a+b+c==a*b*c)
		printf("%d%d%d\n",a,b,c);
	}
}
int main()
{	printf("结果为\n");
	solve();
}
```
