## 求奇偶数个数

编程题

### 题目描述

程序要读入一系列正整数数据，输入-1表示输入结束，-1本身不是输入的数据。程序输出读到的数据中的奇数和偶数的个数。

### 输入格式：

一系列正整数，整数的范围是（0,100000）。

### 输入样例：

```
9 3 4 2 5 7 -1
```

### 输出样例：

```
奇数个数为：4 偶数个数为：2
```

任务与要求

运行程序，使其通过所有检查

# 答案
```c
#include<stdio.h>
int main(){
    int n;
    int i=0,j=0;
    scanf("%d ",&n);
    while(n!=-1){
    if(n%2==1){
        i++;
    }else{
        j++;
    }
    scanf("%d",&n);
}
    printf("奇数个数为：%d 偶数个数为：%d",i,j);
    return 0;
}
```
