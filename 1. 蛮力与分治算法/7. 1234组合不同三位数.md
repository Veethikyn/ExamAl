## 1234组合不同三位数

编程题

### 题目描述

编写程序，判断由1,2,3,4四个数字能组成多少个互不相同且无重复数字的三位数？输出这些数。每输出五个数换行。

### 输出样例：

```
123	124	132	134	142	
143	
共有:6
```

任务与要求

运行程序，使其通过所有检查

# 答案
```c
#include<stdio.h>
int main()
{
    int i,j,k,n=0;
    for(i=1;j<5;i++)
        for(j=1;j<5;j++)
            for(k=1;k<5;k++)
                if(i!=k&&i!=j&&j!=k)
                {
                    printf("%d%d%d\t",i,j,k);
                    if(++n%5==0)printf("\n");
                }
    printf("\n共有:%d\n",n);
    return 0;
}
```
