## 最小公倍数

编程题

### 题目描述

求两个数的最小公倍数。
最小公倍数：最小公倍数两个或多个整数公有的倍数叫做它们的公倍数，其中除0以外最小的一个公倍数就叫做这几个整数的最小公倍数。
例子：
4和5的最小公倍数为20。

### 输入格式：

输入给出两个整数，中间以空格隔开。

### 输入样例：

```
4 5
```

### 输出样例：

```
最小公倍数为20
```

任务与要求

运行程序，使其通过所有检查

# 答案
```c
#include<stdio.h>
int main()
{
 int a=0;int b=0;
 printf("请输入两个数（以空格隔开）：");
 scanf("%d %d",&a,&b);
 int max=(a>b?a:b);//取最大值 
 while(max%a!=0||max%b!=0)//如果不能同时整除 
 {
  max++;
 }
 printf("最小公倍数为%d",max);
 return 0;
}
```
